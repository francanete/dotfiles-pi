# dotfiles-pi

Shareable Pi resources for machines I use.

## Contents

- `extensions/`
- `skills/`
- `prompts/`
- `themes/`

## Install

```bash
pi install git:github.com/francanete/dotfiles-pi
```

Or install a local checkout on one machine:

```bash
pi install ~/Workspace/dotfiles-pi
```

## Notes

- Keep machine-local config in `~/.pi/agent/`.
- Keep secrets, tokens, and per-machine paths out of this repo.
